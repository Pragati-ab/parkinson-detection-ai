About

Parkinson.AI

Parkinson.AI is a machine learning-based diagnostic support system that analyzes 22 biomedical voice features to detect early signs of Parkinson’s disease. The model uses a Support Vector Machine trained on the UCI Parkinson’s dataset and achieves 92.3% accuracy. The goal of this project is to demonstrate how AI can assist in early medical screening.
